# Muntazir
This is a download manager written in c++. (in progress)

##### Disclaimer
> This is a personal project and the purpose is to learn. This is not secure to use at all.

#### Done :
- Set up libcurl and understand the basic functionality
- Calculate the progress status
- Download from http/https urls

#### Known Bugs:
- Doesn't work for some file formats
- Https links don't work sometimes. I don't know the issue yet but it is probably some issue related to the version of libcurl
